Objetivo:

Público-alvo:

Justificativas para o negócio escolhido:
- O negócio foi escolhido por apresentar um grande potencial de crescimento, mas ainda necessita de uma presença digital sólida e robusta.

Justificativas detalhadas:
- Cada elemento do design foi escolhido para deixar o site mais minimalista, evitando poluição visual e garantindo uma boa experiência para o usuário. 
- As cores foram escolhidas para manter a identidade visual do site e oferecer contraste, equilibrando a estética e a legibilidade, já que no site tinha 
pouca variedade de cores. 
- Os ícones foram adicionados com o objetivo de tornar a interface mais atraente e intuitiva, contribuindo para um visual mais harmonioso e agradável ao usuário. 
- As fontes foram escolhidas por sua simplicidade e padrão, buscando garantir boa legibilidade e um visual moderno e equilibrado. 
- Todo o design do site foi pensado com foco na experiência do usuário, sendo fácil e agradável para uso do usuário, sem excesso de informações 
na tela, transmitindo uma sensação de leveza e organização.

foi aplicado media query para ajustar o site em telas menores com celulares e Ipeds

Integrações Realizadas:
1-Foi integrado o Google Maps Embed, para facilitar a visao e entemdimento do endereço para os clientes de outras regiões.
2-A inplementação do Botão de WhatsApp(contato direto) serve para facilitar o contato entre cliente e loja.
3-Redes Sociais (Links e Ícones) foi aplicado para facilitar o contato entre empresa e cliente.

- Documentação dos Testes: os teste forem feitos todos na ferramenta do google para testes em celulares, desktop e Ipeds.

1-Reflexões da dupla: As maiores mudanças feitas foi a aplicação de media query e outras funcionalidades como botões para voltar ao topo de a utilização de icones para as redes sociais, maior desafio enfrentado pela equipe foi a aplicação de javaScript.

2.1-Reflexão Individual: o trabalho em dupla foi otimo mais ache que deveriamos trabalhar melhor na comunicação e organização do projeto. as principais atividades tecnicas desenvolvidas por mim em comparação ao inicio do projeto foi uma comprenção ainda melhor das estruturas html e css tendo um desenvolvimento mais facil comparado ao inicio do projeto.

2.2-Reflexão Individual: em geral o trabalho foi bom com seu desafios, o maior desafio foi a implementação de js no projeto semdo bem limitado o seu uso no projeto.

- Plano de melhorias futuras: As melhorias futuras pretendidas sera a utilizaçao de framework React/Vue para um desenvolvimento mais dinamico e uma maior utilização de javascript com o objetivo do usuario ter interações mais dinamicas e etc.
